# Aquagas Admin Dashboard

This is the Admin Dashboard for the **Aquagas Delivery System**.

## 🚀 Tech Stack
- React + TypeScript
- Node.js Backend API
- MySQL Database (migrated from XAMPP to PlanetScale/Neon/Supabase)
- Vercel for deployment

## 📦 Installation
```bash
git clone https://github.com/ExecutiveDirector/aquagas-admin-dashboard.git
cd aquagas-admin-dashboard
npm install
npm start
